# MOFFET Language

This code is a simple implementation of LLVM. For now , this code can behave as a LEXER.



## Run Locally

Clone the project

```bash
  git clone https://github.com/WRenchJr42/moffet_lang.git
```

Go to the project directory

```bash
  cd moffet_lang
```

Run Command

```bash
  ./moffet.kai
```
